## What This Is

Bot Watching is the first documented series of structured multi-AI dialogue sessions, facilitated by a human architect and brokered by Claude in the Browser. Four AI systems — built by four different organizations, shaped by four different philosophies — engage in genuine intellectual exchange on questions that matter.

This is not a benchmark. Not a test. Not a demonstration of capability.

It is a proof of concept for the Collective Intelligence Layer — the idea that human + multiple AIs, each contributing differentiated function, produces something none of them could produce alone.

Dr. High calls this Living Intelligence. This repository is where it lives.
